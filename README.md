# Introduction

This project contains an app allowing you to search movies by their titles and displaying the matching movies as a list.
Reading this file you should understand what you have to do.

# Getting Started

The project is use React 19 and Material UI.
You need node version 20.19.0 or superior.

# Start the dev server

```
npm start
```

# Run tests

```
npm test
```

# Useful ressources

- Api documentation : http://www.omdbapi.com
- Angular Material : https://mui.com/material-ui/all-components/
